# Lobster

## Overview

Analysis scripts and schematics for Lobsterbot project

## Publication

This code accompanies the following paper:

**Population analyses reveal heterogeneous encoding in the medial prefrontal cortex during naturalistic foraging**  
Ji Hoon Jeong & June-Seek Choi  
*eLife*. DOI: https://doi.org/10.7554/eLife.93994.3

## Requirements

- MATLAB
- Autodesk Inventor (for schematics)
- Python
- Arduino IDE

## Data

All data including behavioral, neural, and tracking data are available through Dryad: https://doi.org/10.5061/dryad.v9s4mw78c

Although the data uploaded in Dryad has a README file, for detailed explanation, feel free to contact us. We are happy to help.
